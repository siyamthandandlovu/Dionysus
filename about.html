About Dionysus

This website is a component of a broader project in which I, Siyamthanda Ndlovu , have taken the front-end pages I developed and ensured their functionality without the backend infrastructure. The website now showcases sample data.

The original Dionysus project was a full-stack application that used a NodeJS API to perform CRUD applications on an SQL database.

It was completed as the final assignment in a second year Computer Science course at the University of Pretoria (COS 221 Intro to Databases).

For further details on the original Dionysus Project, please refer to the repository Dionysus. To access the repository for this specific website, please visit: Dionysus by Siyamthanda Ndlovu
